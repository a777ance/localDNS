Statement Gallery

Mockups of the monthly statements the stack produces — what we're aiming the generator toward. Each opens in your browser and prints cleanly to PDF for mailing. Two sides: the per-household client statement, and the operator-side Alliance Member Portfolio that rolls a whole book of homes into one view.

These are mockups. Query / block / uptime / latency / VPN figures and the "Handled For You" log are real-data-ready today; the volume-by-category sections need a flow-accounting layer first. See the README for the honest data-sourcing breakdown.

Tip — install this like an app: on a phone, open this page and choose Add to Home Screen; on a laptop (Chrome/Edge), click Install in the address bar. It opens full-screen and works offline — no app store needed.
